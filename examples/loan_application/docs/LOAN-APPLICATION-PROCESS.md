# LOAN APPLICATION PROCESS

- Stage 01: PRE-QUALITIFICATION
    1. The borrower provides basic information and authorizes a Soft Pull
    2. The Loan Officer (LO) transfer Input Data into the Loan Origination System (LOS)
    3. The LO Request a Soft Pull from the Credit Bureau:
        - The LOS connects to the Credit Bureau (Experian, Equifax, TransUnion).
        - The LO selects Soft Credit Report / Pre-Qual Report instead of a hard pull."
    4. The LO receive and review the Soft Pull Report
    5. Assess Creditworthiness
    6. Verify Income & Employment Consistency
    7. Discuss Credit Findings with Borrower and Provide Initial Consultation & Strategic Advice
- Stage 02: PRE-APPROVAL
    1. Determine Loan Eligibility
    2. Provide Pre-Qualification Estimate
    3. Address Credit or Documentation Gaps
    4. Documents collections for Pre-Approval
    5. Runs AUS (Desktop Underwriter or Loan Prospector) with hard pull
    6. Issue the Pre-Approval Letter
- Stage 03: PURCHASE OFFER
    1. Property Information
    2. Purchase offer
    3. Purchase aggreement
- Stage 04: OFFICALLY LOAN APPLICATION SUBMISSION
    1. Uploads Signed Purchase Agreement
    2. Document Clollection
    3. Generate Loan Estimate/Disclosures
- Stage 05: UNDERWRITING SUBMISSION
    1. Pulls credit and submits data through Desktop Underwriter (DU)
    2. Order Appraisal & Title Work
    3. Processing & Verifications
    4. Submit to Underwriting
    5. Conditional Approval
    6. Satisfy Conditions
    7. Clear to Close (CTC)
- Stage 06: CLOSING
    1. Closing Disclosure (CD)
    2. Funding
    3. Closing & Signing
